HTML Images

Images can improve the design and the appearance of a web page.

HTML Images Syntax

The HTML img tag is used to embed an image in a web page

Images are not technically inserted into a web page; images are linked to web pages. The img tag creates a holding space for the reference image.

The img tag is empty, it contains attributes only, and does not have a closing tag.

the img tag has two required attributes:

  • src - Specifies the path to the image
  • alt - Specifies an alternative text for the image
Syntax

<img src="url" alt="alternativeText">

The src Attributes

The required src attributes specifies the path (URL) to the image.

Note: When a web page loads, it is the browser, at that moment, that gets the image from a web server and inserts it into the page. Therefore, make sure that the image actually stays in the same spot in relation the the web, otherwise your visitors will get a broken link icon. The broken link icon and the alt text are show if the browser cannot find the image.

Example

<img src="img_chania.jpg" alt="Flowers in Chania">

The alt attributes

The required alt attributes provides an alternate texto for an image, if the user for some resaon cannot view it (because of slow connection, an error in the src attributes, or if the user uses a screen reaer).

The value of the alt attributes should escribe the image:

Example

<img src="wrongname.gif" alt="Flowers in Chania">

If a browser cannot find an image, it will display the value of the alt attributes.

Image Size - Width and Height

You can use the style attributes to specify the width and height of an image

<img src="img_girl.jpg" alt="Black bird with open wings" style="width:500px;height:600px;">
<img src="img_girl.jpg" alt="Black bird with open wings" width="500" height="600">

The width and height attributes always define the width and height of the image pixels.

Note: Always specify the width and height of a image. If width and height are not specified, the web page might flicker while the image loads.

Width and Height, or Style?

the width, height, and style attributes are all valid in HTML.

However, we suggest using the style attribute. It prevents styles sheets from changing the size of images.

Images in Another Folder

If you have images in a sub-folder, ypu must include the folder name in the src sttribute:

Example

<img src="/images/html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">

Images on Another Server/Website

Some web sites point to an image on another server.

To point to an image on another server, you must specify an absolute (full) URL in the src attribute.

Example

<img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3Schools.com">

Notes on external images: External images might be under copyright. If you do not get permission to use it, you may be in violation of copyright laws. In addition, you cannot control external images; they can suddenly be removed or changed.

Animated Image

Example

Image as a Link

To use an image as a link, put the img tag inside the a tag:

Exemple

<a href="default.asp"> <img src="smiley.gif" alt="HTML tutorial" style="width:42px;height:42px;"> </a>

Image Floating

Example

<p> <img src="#" alt="#" style="float:right;width:50px;height:50px;"> The image will float to the right of the text. </p>

<p> <img src="#" alt="#" style="float:left; width:50px; height:50px;"> The image will float to the left of the text.

Chapter Summary

  • Use the HTML img element to define an image
  • Use the HTML src attribute to define the URL of the image
  • Use the HTML alt attribute to define an alternate text for an image, if it cannot displayed.
  • Use HTML b.redwidth and height attributes or the CSS width and height properties to define the size of the image
  • Use the CSS float property to let the image float to the left or to the right

Note: Loading large images takes time, and can slow down your web page. Use images carefully.
